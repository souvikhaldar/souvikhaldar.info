Sorting is the process of organising objects according to their values in certain way, either increasing or decreasing. There are several ways in which we can *sort* objects, but what matters most is how **efficient** is the sorting algorithm. 
In this series of articles I'm going to explain several algorithms to sort and implement them in code in Python programming language. 
Without much ado, lets begin the first article in this series, with the very simple *Selection Sort* algorithm.
